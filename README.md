# Python-kode-masteroppgave
 
Dette repositoriet inneholder Python skript og relevante data brukt i masteroppgaven "Vurdering av unngåtte utslipp for ulike orienteringer av solcelleanlegg ved bruk av en timeoppløst utslippsfaktor for kraftnettet" 
 
## Mappestruktur
 
- `Data utslippsintensitet/`: Timesoppløst utslippsintensitet for europeisk strømmiks 2015–2024
- `TMY data/`: EPW/TMY filer for Oslo, Trondheim, Tromsø, Paris og Roma
- `Python script/`: Python skript for PV-simulering, SARIMA-prognose og plot av figurer
- `requirements.txt`: Liste over nødvendige pakker
- `README.md`: Denne dokumentasjonen
## Kom i gang
 
1. Klon repoet:
   ```
   git clone https://github.com/KarlEinarChristiansen/Python-kode-masteroppgave.git
   ```
2. Installer nødvendige pakker:
   ```
   pip install -r requirements.txt
   ```
